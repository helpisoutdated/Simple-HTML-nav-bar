Home Data Diri Contoh Website

1. Website E-Commerce

Website e-commerce adalah jenis website yang digunakan untuk menjual produk atau jasa secara online. Contohnya adalah Amazon, Tokopedia, dan Shopee. Website ini memungkinkan pengguna untuk membeli barang secara langsung melalui internet.

2. Website Portfolio

Website portfolio digunakan oleh individu atau perusahaan untuk memamerkan karya mereka. Website ini sering digunakan oleh fotografer, desainer grafis, atau freelancer untuk menunjukkan keterampilan mereka kepada calon klien.

3. Website Blog

Website blog biasanya berisi artikel atau tulisan pribadi yang berbagi pengalaman, opini, atau panduan. Contohnya adalah Medium, Blogger, dan WordPress. Blog sering digunakan untuk membangun audiens atau sebagai sarana untuk mengekspresikan diri.

4. Website Berita

Website berita digunakan untuk menyampaikan informasi terkini kepada masyarakat. Contohnya adalah CNN, BBC, dan Kompas. Jenis website ini memiliki fokus pada berita terkini, artikel opini, dan analisis.
